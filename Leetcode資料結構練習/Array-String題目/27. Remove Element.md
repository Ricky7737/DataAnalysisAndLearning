## 題目意思 
> 給定一個 ```nums```一維陣列 與 ```val```，然後輸出一個數值為 nums 裡有幾個不等於 val 的元素個數

## 解題方式
> 步驟1 => 使用 while迴圈讓 val 去對照 nums 裡的元素，出現相同元素時就從 nums 裡刪除    
> 步驟2 => 計算最終 nums 一維陣列長度

## Python3 程式碼
```
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        # 讓 val 去對照 nums 陣列每個元素
         while val in nums:
            # 如果出現一樣就刪除相同的元素
            nums.remove(val)
        # 回傳一個最終 numd 陣列的長度
        return len(nums)
```
