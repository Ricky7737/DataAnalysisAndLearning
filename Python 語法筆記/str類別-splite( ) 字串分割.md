# str類別-splite( ) 字串分割
> Python 字符串('str') 內建的方式，用於透過字串中的空個分割字符，然後返回對象為分割後字串的 List 列表。
### 舉例 :
```
a = "Hello, I am Ricky"
a = a.splite()
print(a)

output : [apple,banana,orange]
-----------------------------------------
設定分割條件例如使用 ','
b = a.split(',')
print(b)

output: ['apple', 'banana', 'orange']
```
