# 羅吉斯回歸(Logistic Regression)
#### 羅吉斯回歸適合用於 "二元分類" 或 "多元分類"。
### Logistic Regression(邏輯斯回歸) vs Linear Regression(線性回歸)
  * **1.Logistic Regression :** 主要進行分類。透過計算一個概率值，根據閾值（通常是0.5）進行分類預測
  *  **2.Linear Regression(線性回歸)**
